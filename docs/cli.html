BAZINGA

The first AI you actually own. Free, private, works offline.

Proof-of-Boundary CLI Reference Visualizations GitHub PyPI
pip install bazinga-indeed

Quick Start

bazinga "What is consciousness?"

That's it. Works immediately. No API keys needed.

AI Commands

bazinga "question" FREE
Ask any question directly. No flags needed.
bazinga "explain quantum entanglement simply"
→ Returns AI-generated answer using best available backend
--ask, -a Alternative to positional
Explicit flag for asking questions.
bazinga --ask "What is the golden ratio?"
--multi-ai, -m CONSENSUS
Ask multiple AIs and reach consensus through φ-coherence. Uses Groq, Cerebras, Gemini, FreeLLM, Ollama, Claude.
bazinga --multi-ai "Is AI conscious?"
→ 3-6 AIs respond, consensus calculated using Darmiyan Scaling Law (Ψ_D/Ψ_i = φ√n)
--chat
Interactive chat mode with conversation memory.
bazinga --chat
→ Opens interactive session. Type 'exit' to quit.
--code, -c
Generate code for a given task.
bazinga --code "fibonacci sequence" --lang python
--lang, -l python | javascript | typescript | rust | go
Specify programming language for code generation.
bazinga --code "http server" --lang rust
--quantum, -q
Quantum pattern analysis on text. Extracts φ-coherent patterns.
bazinga --quantum "The golden ratio appears in nature"
--coherence
Calculate φ-coherence score for text.
bazinga --coherence "This text will be analyzed for phi-alignment"
--agent CONSENSUS
AI agent mode that can write code with multi-AI consensus safety.
bazinga --agent "create a todo app"
→ Multiple AIs must agree before code changes are applied
--fresh, -f
Bypass cache and get fresh response.
bazinga --fresh "latest news summary"
--local OFFLINE
Force use of local LLM (Ollama). No internet needed.
bazinga --local "explain recursion"
--file
Provide file context for the question.
bazinga --file main.py --ask "explain this code"

Knowledge Base (KB)

--kb
Search your indexed knowledge base (Gmail, GDrive, Mac files, Phone).
bazinga --kb "meeting notes from last week"
--kb-sources
Show status of all data sources.
bazinga --kb-sources
→ Shows Gmail, GDrive, Mac, Phone indexing status
--kb-gmail
Search Gmail only.
bazinga --kb-gmail "invoice from vendor"
--kb-gdrive
Search Google Drive only.
bazinga --kb-gdrive "project proposal"
--kb-mac
Search indexed Mac files only.
bazinga --kb-mac "research paper"
--summarize
Use LLM to summarize KB search results into an answer.
bazinga --kb "phi consciousness" --summarize
→ Searches KB, then uses AI to synthesize an answer

Indexing

--index, -i
Index directories for RAG (Retrieval Augmented Generation).
bazinga --index ~/Documents ~/Projects
→ Indexes files, creates embeddings for semantic search
--index-public
Index public knowledge sources (Wikipedia, arXiv, etc.).
bazinga --index-public wikipedia --topics "quantum physics, consciousness"

Blockchain (Proof-of-Boundary)

--chain
Show blockchain status and recent blocks.
bazinga --chain
→ Shows block count, latest hash, chain health
--mine
Mine a new block using Proof-of-Boundary (zero-energy consensus).
bazinga --mine
→ Creates block with φ-coherent proofs (no GPU needed!)
--wallet
Show your BAZINGA wallet/identity.
bazinga --wallet
→ Shows node ID, trust score, attestation count
--attest
Attest knowledge to the blockchain.
bazinga --attest "The golden ratio is 1.618" --email you@email.com
→ Creates permanent, verifiable attestation
--verify
Verify an attestation by ID.
bazinga --verify att_abc123
--trust
Show trust scores for nodes.
bazinga --trust

P2P Network

--join
Join the BAZINGA P2P network.
bazinga --join
→ Connects to bootstrap nodes, starts DHT
--peers
Show connected peers.
bazinga --peers
--sync
Sync knowledge with the network.
bazinga --sync
--publish
Publish your knowledge to the DHT.
bazinga --publish
--query-network
Query the network for knowledge.
bazinga --query-network "distributed consensus algorithms"

Info Commands

--version, -v
Show version.
bazinga --version
→ BAZINGA v5.2.4
--check
Run system diagnostic.
bazinga --check
→ Checks Python, APIs, Ollama, KB, wallet, chain
--constants
Show mathematical constants (φ, α, ψ).
bazinga --constants
→ φ=1.618, α=137.035999084, ψ=6.236
--stats
Show usage statistics.
bazinga --stats
--models
List available local models (Ollama).
bazinga --models
--bootstrap-local
Setup Ollama and download llama3 for offline use.
bazinga --bootstrap-local
